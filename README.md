## About

A basic portfolio website featuring the Rose Pine colour scheme and built with MDX for ease of writing.

### Built with

The main technologies used to build peterjunpark.com

* [React](https://react.dev)
* [Next.js 14 (App router)](https://nextjs.org)
* [MDX](https://nextjs.org/docs/app/building-your-application/configuring/mdx)
* Remark and Rehype plugins (check `package.json` or `next.config.mjs`)
* [Rose Pine](https://rosepinetheme.com)

## Installation

First, get the things you'll need:

### Prerequisites

* Install [Node.js](https://nodejs.org)
* Install [pnpm](https://pnpm.io/installation)
  ```sh
  npm i -g pnpm
  ```

### Getting started

Now, complete these steps:

1. Clone the repo
   ```sh
   git clone https://github.com/peterjunpark/peterjunpark.com.git
   ```
2. Install dependencies via pnpm
   ```sh
   cd peterjunpark.com
   pnpm i
   ```
3. Start the development server
   ```sh
   pnpm dev
   ```
4. Go to `http://localhost:3000` in your web browser.
